Amazon-SNS-Ruby by cheeyeo

Ruby gem for the new Amazon SNS service.

The Amaze SNS gem is designed to interact with the new Amazon Simple Notification Service. You can create topics; add subscribers; and create and delete topics remotely.

Dependencies

Requires Crack gem; Event Machine; and EM-HTTP-REQUEST gem

Install

sudo gem install amaze-sns
